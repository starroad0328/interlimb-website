# 개발 일지 — 2026-02-16

## 작업 내용

### 1. 프로젝트 초기화
- Next.js 14 (App Router) + TypeScript + Tailwind CSS v3 프로젝트 수동 셋업
- Framer Motion 설치
- `tsconfig.json`, `postcss.config.mjs`, `next.config.mjs`, `.gitignore` 설정

### 2. 디자인 시스템 구성
- `tailwind.config.ts` 커스텀 설정
  - 컬러: `gold(#C5A358)`, `gold-light(#D4AF37)`, `background(#FDFDFD)`, `text(#1A1A1A)`, `grid(#E5E7EB)`
  - 폰트: Inter (본문), Montserrat (디스플레이)
- `globals.css`에 Tailwind base + `glass` 유틸리티 클래스 추가
- `layout.tsx`에 Google Fonts (next/font) + SEO 메타데이터 설정

### 3. Hero 섹션 구현 (v1 → v2)
**v1**: 단순 SVG 선 기반 아이소메트릭 그리드 + 얇은 가지
- 가지와 흙더미가 분리되어 어색함
- 그리드가 중앙에서 벗어남

**v2 (현재)**: 전면 재설계
- `IsometricGrid.tsx`: 입체 타일 (상면 + 측면 depth) + 중앙 금빛 radial glow
- `GoldenTree.tsx`: 가지 + 흙더미 통합 SVG. 12개 유기적 곡선, 금색 그라디언트 + glow filter + highlight 3중 레이어
- `InfoCard.tsx`: 가지 끝에 글래스모피즘 카드 5개 배치
- 모바일에서는 카드 숨김, 텍스트 fallback

### 4. About / Products / Contact 섹션
- `AboutSection.tsx`: 비전/미션 텍스트 + `whileInView` 스크롤 애니메이션
- `ProductsSection.tsx` + `ProductCard.tsx`: NotioClass 카드 (글래스모피즘, hover glow)
- `ContactSection.tsx`: 이메일 CTA 버튼 + GitHub/LinkedIn 아이콘

### 5. Navbar
- 스크롤 시 나타나는 고정 네비게이션 (Hero 영역에서는 숨김)
- 모바일 햄버거 메뉴 + AnimatePresence 토글

### 6. GitHub 배포
- `starroad0328/interlimb-website` 리포지토리 생성 및 push

## 기술적 결정/이슈

| 이슈 | 결정 | 이유 |
|------|------|------|
| Tailwind v4 자동 설치됨 | v3으로 다운그레이드 | Next.js 14와 호환성, `tailwind.config.ts` 기반 설정 방식 유지 |
| `JSX.Element` 타입 에러 | `ReactElement`로 변경 | TypeScript 5.9 + React 18 환경에서 JSX 네임스페이스 미노출 |
| SVG로 샘플 수준 재현 불가 | 향후 이미지 배경 방식 검토 | 3D 금속 질감/조명은 CSS/SVG 한계, 렌더링 이미지 활용이 현실적 |
